# 工程简介
项目为springboot和netty-socketIO 整合
利用到 rabbitmq 作为发布订阅中间件，为了实现分布式部署

│   │               ├── conf
│   │               ├── controller
│   │               ├── listener
│   │               └── service
│   └── resources
│       ├── static
│       │   └── demo
│       └── templates
└── test
